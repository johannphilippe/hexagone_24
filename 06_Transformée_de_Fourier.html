Transformée de Fourier et analyse fréquentielle

La transformée de Fourier généralise le principe de la série de Fourier, en introduisant une méthode d'analyse des fréquences d'un signal.

Elle s'exprime ainsi :

  • F(x)=∫Re−iωtdt=∫Re−i2πftdtF(x) = \int_{\mathbb{R}}e^{-i \omega t}dt = \int_{\mathbb{R}}e^{-i 2 \pi f t}dtF(x)=∫R​e−iωtdt=∫R​e−i2πftdt

Cette fonction transforme une fonction intégrable sur R\mathbb{R}R en une autre fonction qui décrit le spectre fréquentiel de cette dernière.

Petit Rappel d'hier : L'exponentielle complexe Elle est une représentation synthétique des sinus et cosinus. Elle est utilisée notamment pour l'analyse fréquentielle d'un signal.

En signal continu :

  • ejωt=cos(ωt)+jsin(ωt)e^{j \omega t} = cos(\omega t) + jsin(\omega t)ejωt=cos(ωt)+jsin(ωt)
    En signal discret :
  • ejωn=cos(ωn)+jsin(ωn)e^{j \omega n} = cos(\omega n) + jsin(\omega n)ejωn=cos(ωn)+jsin(ωn)

La FFT (Fast Fourier Transform) est une méthode d'analyse EXTRÊMEMENT prosaïque : on évalue la présence de fréquences en multipliant notre signal par ces fréquences.

Implémentation simple

def fft_r2c(sig):
    fft_size = len(sig) 
    result = np.ndarray(shape=(2, fft_size))
    # For each FFT bin (each frequency we try)
    for k in range(fft_size):
        result[0][k] = 0
        result[1][k] = 0
        # Iterate over samples to determine how much signal samples match our sines and cosines
        for n in range(fft_size):
            # Here k is frequency and n is phase
            nn = -2 * pi / fft_size * k * n 
            result[0][k] += cos(nn) * sig[n] 
            result[1][k] += sin(nn) * sig[n]
    return result

Remarques :

  • La taille de la FFT - soit la Rf⋅2Rf \cdot 2Rf⋅2 où RfRfRf représente résolution fréquentielle 2 - est égale au nombre d'échantillons analysés
  • Donc, plus on analyse d'échantillons, plus on a une bonne définition fréquentielle
  • On supprime totalement la définition temporelle dans la FFT

Calcul de la magnitude spectrale

Attention : très simple

La magnitude pour chaque bin FFT (chaque fréquence représentée) est égale à ∣X(k)∣=Re(X[k])2+Im(X[k])2|X(k)| = \sqrt{Re(X[k])^{2} + Im(X[k])^{2}}∣X(k)∣=Re(X[k])2+Im(X[k])2​
Soit la valeur absolue de chaque bin.

Implémentation

def magnitude(fft_sig):
    l = len(fft_sig[0])
    mag = np.zeros(l)
    for k in range(l):
        mag[k] = sqrt( pow(fft_sig[0][k], 2) + pow(fft_sig[1][k], 2) )
    return mag

La magnitude représente l'amplitude absolue de la bande (le bin) dans le domaine fréquentiel.

Calcul de la phase spectrale

ϕ(k)=tan⁡−1(Im(X(k))Re(X(k)))\phi(k) = \tan^{-1}(\frac{Im(X(k))}{Re(X(k))})ϕ(k)=tan−1(Re(X(k))Im(X(k))​)

Modification de la phase spectrale :

  • X′(k)=∣X(k)∣eϕ(k)+ΔϕX'(k) = |X(k)|e^{\phi(k)+\Delta \phi}X′(k)=∣X(k)∣eϕ(k)+Δϕ où Δ\DeltaΔ est un déphasage
  • Inversion de la phase dans la même équation dans Δ\DeltaΔ où ϕ(k)=−ϕ(k)\phi(k) = -\phi(k)ϕ(k)=−ϕ(k)

Quelques opérations

Manipulation des bandes

On peut simplement multiplier une bande (le réel et l'imaginaire) pour modifier sa présence dans le spectre à travers :

  • La multiplication
  • L'addition (avec une autre analyse FFT) - sous réserve que les signaux aient la même résolution spectrale et la même synchronisation
  • Filtrage fréquentiel : on multiplie les coefficients X(k)X(k)X(k) par une fonction spécifique pour supprimer ou atténuer certaines fréquences.
  • Modification de phase : ajouter un déphasage constant ou une modulation spécifique à l'angle de chaque composante X(k)
  • Compression ou expansion spectrale :
    • Réduire ou élargir la largeur spectrale en regroupant plusieurs fréquences en une seule pour simplifier le spectre ou pour compression de données
    • Étirer le spectre en dilatant l'échelle fréquentielle : pitch shifting
  • Manipuler les coefficients X(k)X(k)X(k) dans le domaine fréquentiel puis appliquer une transformée inverse pour obtenir une version modifiée du signal temporel (élimination de bruit fréquentiel spécifique, création d'écho)
  • Time stretching (On étire l'échelle temporelle d'un signal en maintenant ses fréquences) : STFT (on voit ça après)
  • Morphing spectral : interpolation des magnitudes par exemple
  • Générer un signal synthétique basé sur les fréquences dominantes d'un signal analysé
  • Inversion spectrale (inverser l'ordre des coefs)
  • Effet robotique : éliminer les phases (phase constante à toutes les fréquences )
  • Appliquer des transformations non-linéaires aux amplitudes et phases (compression logarithmique, élévation à une puissance)
  • Transposition fréquentielle : décaler les fréquences en multipliant chaque kkk par un facteur fff
  • Convolutions
  • Machine learnning
    • Extraire des éléments pertinents comme les formants, la f0f_0f0​, le bruit
    • Compression perceptuelle : réduire les coefficients en supprimant ceux en dessous d'un seuil par ex

Estimation de la densité spectrale de puissance

La densité spectral de puissance peut être exprimée comme :

  • En continu :
    • Sx(f)=lim⁡T→∞1T∣∫−T/2T/2x(t)e−j2ϕftdt∣2S_x(f) = \lim_{T \to \infty} \frac{1}{T} | \int_{-T/2}^{T/2} x(t)e^{-j2\phi f t}dt|^{2}Sx​(f)=limT→∞​T1​∣∫−T/2T/2​x(t)e−j2ϕftdt∣2
  • En discret :
    • Sx(f)=1N∣∑n=0N−1x(n)e−j2πfn∣2S_x(f) = \frac{1}{N}|\sum_{n=0}^{N-1}x(n)e^{-j2 \pi f n}|^{2}Sx​(f)=N1​∣∑n=0N−1​x(n)e−j2πfn∣2

En fait c'est simplement un carré de la magnitude, qu'on peut éventuellement pondérer (ou moyenner) pour avoir une valeur normalisée.

La densité spectrale de puissance peut s'exprimer comme suit :

def psd(fft_sig, sr):
    l = len(fft_sig[0])
    power = np.ndarray(l)
    mags = magnitude(fft_sig)

    psd = np.ndarray(l)
    for n in range(l):
        # PSD can be described as the square of the magnitude/amplitude, weighted by sr an size of fft
        res = (2 / ( sr * l)) * np.abs(complex(fft_sig[0][n], fft_sig[1][n])) ** 2
        psd[n] = res
    return psd

La convolution

Rappel : une convolution dans le domaine temporel est une simple multiplication des bandes dans le domaine fréquentiel. Ainsi :

  • x(n)∗h(n)=X(k)⋅H(k)x(n) * h(n) = X(k) \cdot H(k)x(n)∗h(n)=X(k)⋅H(k)

On peut donc déduire qu'un système peut être caractérisé comme :

  • Y(k)=X(k)∗H(k)Y(k) = X(k) * H(k)Y(k)=X(k)∗H(k) En dehors de l'aspect purement computationnel (moins coûteux qu'une convolution temporelle), cette expression nous permet d'obtenir les deux suivantes :
  • X(k)=Y(k)H(k)X(k) = \frac{Y(k)}{H(k)}X(k)=H(k)Y(k)​
  • H(k)=Y(k)X(k)H(k) = \frac{Y(k)}{X(k)}H(k)=X(k)Y(k)​

On peut ainsi déterminer la fonction de transfert à partir de l'entrée et de la sortie d'un système dans le domaine fréquentiel. Idem, on peut déterminer l'entrée à partir de la sortie et de la fonction de transfert.

Transformée de Fourier inverse (IFFT)

La transformée de Fourier inverse a plusieurs étapes :

  • une conjuguée complexe du signal FFT
  • Le même algorithme que la FFT
  • Une autre conjuguée complexe sur le résultat
  • On garde le signal réel final

Cela nous donne une reconstruction du signal en domaine temporel.

Conceptions dans le domaine fréquentiel

Spectre variant dans le temps

L'analyse spectrale ne représente - par défaut - que des fréquences. Comment représenter le spectre d'un signal dont les composantes fréquentielles évoluent dans le temps ?

Exemple d'une sinusoide qui varie entre deux fréquences

STFT - la short time fourier transform

Ou tranformée de fourier en temps réel. Pour pallier au problème que la FFT (ou DTFT) ne représente que la fréquence, on applique la FFT sur des buffers du signal, représentant une fenêtre de temps.

C'est assez compliqué car il existe deux contraintes :

  • La bufferisation (AGAIN)
  • La superposition (!attention!)

Fenêtrage

Fenêtre rectangulaire

  • Temps continu
    • w(t)={1, si −T/2≤t≤T/20, sinon w(t) = \begin{cases} 1, & \text{ si } -T/2 \le t \le T/2\\0, & \text{ sinon } \end{cases}w(t)={1,0,​ si −T/2≤t≤T/2 sinon ​
  • Temps discret
    • w(n)={1, si 0≤n≤N−10, sinon w(n) = \begin{cases} 1, & \text{ si } 0 \le n \le N - 1\\0, & \text{ sinon } \end{cases}w(n)={1,0,​ si 0≤n≤N−1 sinon ​

La fenêtre rectangulaire n'applique aucune pondération. Elle conserve la totalité du signal. Totalement inutile en temps discret. Probablement pas très utile en continu non plus.

  • Propriétés :
    • Pas de superposition idéalement - introduisant des discontinuités à chaque segment.
    • Artefacts spectraux (effet de Gibbs)

Fenêtre de Hanning (ou Hann)

  • Temps continu
    • w(t)=0.5(1−cos(2πtT)),−T/2≤t≤T/2w(t) = 0.5 (1 - cos( \frac{2 \pi t}{T})), -T/2 \le t \le T/2w(t)=0.5(1−cos(T2πt​)),−T/2≤t≤T/2
  • Temps discret
    • w(n)=0.5(1−cos(2πnN−1)),0≤t≤N−1w(n) = 0.5 (1 - cos( \frac{2 \pi n}{N - 1})), 0 \le t \le N - 1w(n)=0.5(1−cos(N−12πn​)),0≤t≤N−1

La fenêtre de Hanning réduit les oscillations de fuite spectrale grâce à un poids (pondération) en cosinus. Ses propriétés :

  • Superposition :
    • 50 % de recouvrement permet une reconstruction parfaite dans le domaine temporel
    • Mais on peut aller juqu'à 75% pour réduire davantage les discontinuités
  • Sensible à l'augmentation excessive de la superposition (surpondération des extrêmités)

Fenêtre de Hamming

  • Temps continu :
    • w(t)=0.54−0.46cos(2πtT),−T/2≤t≤T/2w(t) = 0.54 - 0.46 cos(\frac{2 \pi t}{T}), -T/2 \le t \le T/2w(t)=0.54−0.46cos(T2πt​),−T/2≤t≤T/2
  • Temps discret :
    • w(n)=0.54−0.46cos(2πnN−1),0≤t≤N−1w(n) = 0.54 - 0.46 cos(\frac{2 \pi n}{N -1}), 0 \le t \le N - 1w(n)=0.54−0.46cos(N−12πn​),0≤t≤N−1

La fenêtre de Hamming ressemble à celle de Hanning, mais avec une atténuation des lobes secondaires plus forte. Propriétés :

  • Superposition :
    • 50 % de recouvrement permet une reconstruction parfaite dans le domaine temporel
    • Mais on peut aller juqu'à 75% pour réduire davantage les discontinuités
  • Idem que Hanning

Fenêtre de Blackman

  • Temps continu :

    • w(t)=0.42−0.5cos(2πtT)+0.08cos(4πtT),−T/2≤t≤T/2w(t) = 0.42 - 0.5 cos(\frac{2 \pi t}{T}) + 0.08 cos(\frac{4 \pi t}{T}), -T/2 \le t \le T/2w(t)=0.42−0.5cos(T2πt​)+0.08cos(T4πt​),−T/2≤t≤T/2

  • Temps discret :

    • w(n)=0.42−0.5cos(2πnN−1)+0.08cos(4πnN−1),0≤t≤N−1w(n) = 0.42 - 0.5 cos(\frac{2 \pi n}{N - 1}) + 0.08 cos(\frac{4 \pi n}{N - 1}), 0 \le t \le N - 1w(n)=0.42−0.5cos(N−12πn​)+0.08cos(N−14πn​),0≤t≤N−1

    La fenêtre de Blackman offre une meilleure suppression des lobes secondaires au prix d'une largeur du lobe principal accrue. Propriétés :

    • Superposition :
      • 50 % de recouvrement permet une reconstruction parfaite dans le domaine temporel
      • Mais on peut aller juqu'à 75% pour réduire davantage les discontinuités
    • Largeur accrue du lobe principal : résolution spectrale diminuée

Fenêtre de Kaiser

  • Temps continu
    • w(t)=I0(β1−(2tT)2)I0(β),−T/2≤t≤T/2w(t) = \frac{I_0(\beta \sqrt{1 - (\frac{2t}{T})^{2} } )}{I_0(\beta)}, -T / 2 \le t \le T/2w(t)=I0​(β)I0​(β1−(T2t​)2​)​,−T/2≤t≤T/2
  • Temps discret
    • w(n)=I0(β1−(2nN−1)2)I0(β),0/2≤t≤N−1w(n) = \frac{I_0(\beta \sqrt{1 - (\frac{2n}{N-1})^{2} } )}{I_0(\beta)}, 0 / 2 \le t \le N - 1w(n)=I0​(β)I0​(β1−(N−12n​)2​)​,0/2≤t≤N−1

où I0I_0I0​ est la fonction de Bessel modifiée de premier ordre, et β\betaβ est un paramètre réglable qui contrôle la largeur de la fenêtre.

  • Fonction de Bessel modifiée de premier ordre :
    • I0(x)=∑k=0∞1(k!)2(x2)2kI_0(x) = \sum_{k = 0}^{\infty} \frac{1}{(k!)^{2}}(\frac{x}{2})^{2k}I0​(x)=∑k=0∞​(k!)21​(2x​)2k

Propriétés :

  • Superposition :
    • Dépend de β\betaβ : généralement 50% est suffisant, mais peut aller jusqu'à 75%.
  • Complexité du paramétrage β\betaβ
  • Risque de perte energétique si le recouvrement est trop faible

LA fréquence d'une bande

On peut déterminer la fréquence d'une bande à partir de la formule suivante :

  • k⋅fsN\frac{k \cdot f_s}{N}Nk⋅fs​​ où kkk est l'index du bin, fsf_sfs​ la fréquence d'échantillonnage et NNN la taille de la FFT.
