Les fonctions d'intercorrélation et d'autocorrélation

La fonction d'intercorrélation (corrélation croisée) mesure la similarité entre deux signaux décalés dans le temps. Elle est utilisée pour identifier les relations et similitudes temporelles entre deux signaux. Elle mesure combien un signal xxx ressemble à une version décalée d'un autre signal yyy. Le pic de cette fonction est atteint pour une certaine valeur de décalage.

Définition mathématique

Domaine Continu

Pour deux signaux continus x(t)x(t)x(t) et y(t)y(t)y(t), l'intercorrélation est définie comme :

  • Rxy(τ)=∫−∞+∞x(t)⋅y(t+τ)dtR _{xy}(\tau) = \int_{-\infty}^{+\infty} x(t) \cdot y(t + \tau) dtRxy​(τ)=∫−∞+∞​x(t)⋅y(t+τ)dt
    où :
  • RxyR _{xy}Rxy​ est la fonction d'intercorrélation
  • τ\tauτ est le décalage temporel, aussi nommé lag
  • x(t)x(t)x(t) et y(t)y(t)y(t) sont les deux signaux comparés

Domaine discret Pour deux signaux continus x(n)x(n)x(n) et y(n)y(n)y(n), l'intercorrélation est définie comme :

  • Rxy(k)=∑n=−∞+∞x(n)⋅y(n+k)R _{xy}(k) = \sum_{n=-\infty}^{+\infty}x(n) \cdot y(n+k)Rxy​(k)=∑n=−∞+∞​x(n)⋅y(n+k)
  • où kkk représente le décalage entre les échantillons

Simplification

Pour chaque délai étudié entre les deux signaux, on multiplie notre signal (tous les échantillons des deux signaux, en discret) et on réalise une somme, ou une moyenne. De fait, on obtient un signal de mesure qui indique pour chaque délai étudié si la correspondance est importante ou non.

Utilisations

L'intercorrélation - en plus d'être un fondement d'autres théories plus abouties, est utilisée opur un certain nombre d'analyses de signal :

  • Détection de pattern et de similarités (correlation_pattern.py)
  • Alignement temporel : synchronisation de signaux (télécommunication par ex)
  • Analyse de délai (et correction)
  • Analyse de fréquence / de vitesse

Autocorrélation

L'autocorrélation est un cas particulier de l'intercorrélation. Il s'agit d'une intercorrélation du signal avec lui-même, tel que :

  • Rx(k)=∑n=−∞+∞x(n)⋅x(n+k)R _{x}(k) = \sum_{n=-\infty}^{+\infty}x(n) \cdot x(n+k)Rx​(k)=∑n=−∞+∞​x(n)⋅x(n+k)

Elle est utilisée, entre autres pour de l'analyse de fréquence fondamentale f0f _0f0​ sur des signaux déterministes.

Défauts

En informatique, cette méthode peut être coûteuse car elle implique de calculer tous les échantillons pour chacun des délais étudiés.

Le produit de convolution

Le produit de convolution (couramment : la convolution) est un outil semblable à l'intercorrélation - qui a cependant d'autres caractéristiques. Pour deux signaux continux x(t)x(t)x(t) et h(t)h(t)h(t) :

  • (x∗h)(t)=∫−∞+∞x(t−τ)h(t)dτ=∫−∞+∞x(t)g(t−τ)dt(x * h)(t) = \int_{-\infty}^{+\infty} x(t - \tau)h(t)d\tau = \int_{-\infty}^{+\infty}x(t)g(t - \tau)dt(x∗h)(t)=∫−∞+∞​x(t−τ)h(t)dτ=∫−∞+∞​x(t)g(t−τ)dt
    Pour deux signaux discrets :
  • (x∗h)(n)=∑m=−∞+∞x(n−m)h(m)=∑m=−∞+∞x(m)h(n−m)(x * h)(n) = \sum_{m=-\infty}^{+\infty} x(n - m)h(m) = \sum_{m=-\infty}^{+\infty}x(m)h(n-m)(x∗h)(n)=∑m=−∞+∞​x(n−m)h(m)=∑m=−∞+∞​x(m)h(n−m)

Dans les deux cas, on nomme :

  • xxx le signal d'entrée
  • hhh la réponse impulsionnelle
  • τ\tauτ ou mmm la variable d'intégration représentant le décalage temporel

La convolution mesure de combien la forme d'un signal est modifiée par un autre signal.

Propriétés

  1. Commutativité : x∗h=h∗xx * h = h * xx∗h=h∗x
  2. Associativité : (x∗h)∗h=x∗(h∗g)(x * h) * h = x * (h * g)(x∗h)∗h=x∗(h∗g)
  3. Distributivité : x∗(h+g)=x∗h+x∗gx * (h + g) = x * h + x * gx∗(h+g)=x∗h+x∗g
  4. Impulsion : le dirac laisse la fonction inchangée x(n)∗δ(n)=x(n)x(n) * \delta(n) = x(n)x(n)∗δ(n)=x(n)
  5. Une convolution dans le domaine temporel revient à une multiplication dans le domaine fréquentiel

Applications de la convolution

La convolution est très largement utilisée dans le traitement du signal :

  1. Filtrage (FIR) : où xxx est un signal d'entrée et hhh une réponse impulsionnelle (exemple python)
  2. Transformation : application des caractéristiques fréquentielles d'un signal à un autre (clonage acoustique, réverbération etc) : exemple des IR de l'IRCAM
  3. Analyse de système : la convolution permet de décrire la sortie d'un système y(t)y(t)y(t) en fonction de l'entrée x(t)x(t)x(t) et de la réponse impulsionnelle h(t)h(t)h(t)

Exemple manuel discret

  • x(n)=[2,1,2]x(n) = [2, 1, 2]x(n)=[2,1,2]
  • h(n)=[1,−1,2]h(n) = [1, -1, 2]h(n)=[1,−1,2]
  1. Renversement de h(n)h(n)h(n) pour obtenir h(−k)h(-k)h(−k)
  2. Décalage de h(−k)h(-k)h(−k) de nnn, ce qui donne h(n−k)h(n-k)h(n−k)
  3. On multiplie les termes qui se chevauchent et on les additionne pour chaque nnn
  • xxx a une longueur de 3, hhh a une longueur de 3.
  • La longueur de (x∗h)(x * h)(x∗h) sera 3+3−13 + 3 - 13+3−1 soit 555
  • Itération sur chaque nnn
    • On itère sur les k pour h(n−k)h(n-k)h(n−k)
    • Pour n=0n=0n=0 on a donc juste k=0k = 0k=0, donc y(0)=x(0)∗h(0−0)=2∗1=2y(0) = x(0) * h(0-0) = 2 * 1 = 2y(0)=x(0)∗h(0−0)=2∗1=2
    • Pour n=1n=1n=1 on a donc juste k=0k = 0k=0 et k=1k = 1k=1, donc y(1)=(x(1)∗h(1−0))+(x(1)∗h(1−1))=2∗−1+1∗1=−1y(1) = (x(1) * h(1-0)) + (x(1) * h(1 - 1)) = 2 * -1 + 1 * 1 = -1y(1)=(x(1)∗h(1−0))+(x(1)∗h(1−1))=2∗−1+1∗1=−1

Analyse spectrale

L'analyse spectrale est une méthode utilisée pour étudier les différentes composantes fréquentielles d'un signal. Elle permet de comprendre comment l'énergie ou la puissance d'un signal se répartit entre différentes fréquences.

La série de Fourier Tout signal périodique peut se décomposer en une somme de signaux sinusoïdaux dont les fréquences sont les multiples entiers d'une fréquence fondamentale

Exemples classiques :

  • Un signal carré peut être approximé avec une série de sinusoïdes dont les fréquences sont les multiples impaires d'un fondamental
  • Un signal triangulaire pareil, avec les multiples pairs (excepté les coefficients qui diminuent plus rapidement)
