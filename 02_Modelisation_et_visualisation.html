Python - prototypage, visualisation

D'un point de vue pratique, le prototypage de signaux dans le domaine numérique est le plus souvent fait en Python (ou en Matlab).

import numpy as np
from math import *
import matplotlib.pyplot as plt
import random

f = 2
N = 100
SR = 100
# Création d'un signal sinusoidal
sine = [np.sin(2 * np.pi * f * n / SR) for n in range(N)]
# Création d'un signal aléatoire (bruit)
noise = [random.random() - 0.5 * 2 for n in range(N)]

x = np.arange(0, N, 1)
fig, axs = plt.subplots(2)
axs[0].plot(x, sine)
axs[1].plot(x, noise)
plt.show()

Classification des signaux

Périodicité On dit qu'un signal est périodique si les variations de son amplitude se reproduisent régulièrement, pendant une la durée de l'analyse, au bout d'une période TTT constante.

Déterminisme Un signal peut être déterministe si on peut prédire son amplitude à un instant t. S'il ne l'est pas, il est aléatoire (bruit blanc, gaussien, ou fractal). Attention, cette définition peut être problématique quand on considère des signaux discrets. Le déterminisme en domaine discret renvoie beaucoup à la notion de périodicité, ou de régularité.

Stationnarité On dit qu’un signal aléatoire est stationnaire si ses propriétés statistiques sont invariantes par translation dans le temps Cela se traduit par :
Px(t1),x(t2),...x(tk)=Px(t1−γ),x(t2−γ),...x(tk−γ)\mathcal{P} x(t1), x(t2), ... x(tk) = \mathcal{P} x(t1 - \gamma), x(t2-\gamma), ... x(tk - \gamma)Px(t1),x(t2),...x(tk)=Px(t1−γ),x(t2−γ),...x(tk−γ) On peut dire qu'il s'agit d'un processus stochastique dont les statistiques (moyenne, variance, corrélation etc) ne changent pas dans le temps.

Ergodisme

L'ergodisme - propriété rarement vérifiable - est souvent un postulat, nécessaire pour travailler. Le signal X(t,ω)X(t, \omega)X(t,ω) est dit ergodique si les moyennes temporelles sont des nombres certains - c'est-à-dire qu'elles ont une

Un signal ergodique est un processus stochastique où les moyennes temporelles (calculées sur une seule réalisation du signal) sont égales aux moyennes d'ensemble (calculées sur un grand nombre de réalisations). Ainsi, si un signal est stationnaire ET ergodique, on peut estimer ses statistiques à partir d'une seule réalisation du signal. La moyenne temporelle est alors un nombre certain, car elle est égale à la moyenne d'ensemble.

Transitoire Le diract δ(t)\delta(t)δ(t) est transitoire. Il est parfaitement déterministe, mais n'a aucune périodicité.

Signaux linéaires

Un système est dit linéaire s'il respecte les propriétés suivantes :

  • Additivité - ou superposition : S[x1(t)+x2(t)]=S[x1(t)]+S[x2(t)]S[x_1(t) + x_2(t)] = S[x_1(t)] + S[x_2(t)]S[x1​(t)+x2​(t)]=S[x1​(t)]+S[x2​(t)]
  • Homogénéité - ou proportionnalité : S[a⋅x(t)]=a⋅S[x(t)]S[a \cdot x(t)] = a \cdot S[x(t)]S[a⋅x(t)]=a⋅S[x(t)]

Par exemple, la fonction y(t)=2x(t)y(t) = 2x(t)y(t)=2x(t) est linéaire car :

  • Additivité : y(t)=2[x1(t)+x2(t)]=2x1(t)+2x2(t)y(t) = 2[x_1(t) + x_2(t)] = 2x_1(t) + 2x_2(t)y(t)=2[x1​(t)+x2​(t)]=2x1​(t)+2x2​(t)
  • Homogénéité : y(t)=2[a⋅x(t)]=a⋅(2x(t))y(t) = 2[a \cdot x(t)] = a \cdot (2x(t))y(t)=2[a⋅x(t)]=a⋅(2x(t))

De nombreux systèmes sont linéaires en traitement du signal : certains filtres, convolution (...).

Un système qui ne garantit pas l'additivité et/ou l'homogénéité est dit non-linéaire. Son comportement comprend une part d'autonomie. Ainsi, la linéarité garantit la non-autonomie du système.

  • y(t)=x2(t)y(t) = x^{2}(t)y(t)=x2(t)

  • Pas d'homogénéité : (a⋅x(t))2=a2⋅x2(t)≠a⋅x2(t)(a \cdot x(t))^{2} = a^{2} \cdot x^{2}(t) \neq a \cdot x^{2}(t)(a⋅x(t))2=a2⋅x2(t)=a⋅x2(t)

Signaux temps-invariant

De manière similaire, un signal temps-invariant est un signal dont les caractéristiques ne changent pas au cours du temps. Ses propriétés restent identiques indépendemment du moment où on l'observe. Ainsi un système temps-invariant est un système dont le comportement ne dépend pas du moment où un signal d'entrée lui est appliqué. Un délai temporel de l'entrée entraine donc un délai identique de la sortie.

Ainsi, si :

  • x(t)→Sy(t)x(t) \to^{S} y(t)x(t)→Sy(t)
    Alors :
  • x(t−t0)→Sy(t−t0)x(t - t0) \to^{S} y(t - t0)x(t−t0)→Sy(t−t0)

De nombreux filtres linéaires réagissent de manière invariante dans le temps. De même un amplificateur du signal est temps-invariant.

LTI

LTI signifie "Linear time invariant" - ils combinent les deux dernières catégories : on va voir plus tard que ces systèmes représentent une grande classe de signaux.
